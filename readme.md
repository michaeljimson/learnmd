> 表格  

Markdown使用`|`来分隔不同的单元格，使用`-`来分隔表头和其他行  
```
|  表头  |  表头  |  表头  |
|  ---   |  ---  |   ---  |
|  内容  |  内容  |  内容  |
|  内容  |  内容  |  内容  |
```
|  表头  |  表头  |  表头  |
|  ---   |  ---  |   ---  |
|  内容  |  内容  |  内容  |
|  内容  |  内容  |  内容  |
  
对齐方式:
 
- `-:`设置内容和标题栏右对齐  
- `:-`设置内容和标题栏左对齐  
- `:-:`设置内容和标题栏居中对齐  
  
如：
```
|  表头  |  表头  |  表头  |
|  :---   |  ---:  |   :---:  |
|  内容  |  内容  |  内容  |
|  内容  |  内容  |  内容  |
```
|  表头  |  表头  |  表头  |
|  :---   |  ---:  |   :---:  |
|  内容  |  内容  |  内容  |
|  内容  |  内容  |  内容  |
